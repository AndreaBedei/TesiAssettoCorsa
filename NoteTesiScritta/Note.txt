I dati utilizzati sono:
\section*{Dataset Features}

The following features were selected from both the physics and graphics interfaces of the simulator to build the dataset for grip prediction:

\begin{itemize}
    \item \textbf{gas}: Throttle pedal input (range: 0.0 to 1.0).
    \item \textbf{brake}: Brake pedal input (range: 0.0 to 1.0).
    \item \textbf{rpm}: Engine revolutions per minute.
    \item \textbf{steer}: Steering input angle.
    \item \textbf{speed}: Vehicle speed in km/h.
    \item \textbf{g\_force}: 3D vector representing forces acting on the car body.
    \item \textbf{wheel\_slip}: Amount of slip for each of the four tires.
    \item \textbf{pressure}: Tire pressure for each wheel.
    \item \textbf{tyre\_temp}: Core temperature of each tire.
    \item \textbf{air\_temp}: Ambient air temperature.
    \item \textbf{road\_temp}: Road surface temperature.
    \item \textbf{yaw\_rate}: Yaw angular velocity (rotation around the vertical axis).
    \item \textbf{slip\_angle}: Slip angle of the tires (angle between wheel heading and actual direction).
    \item \textbf{slip\_ratio}: Slip ratio (longitudinal slip of each tire).
    \item \textbf{brake\_pressure}: Pressure applied on the braking system.
    \item \textbf{brake\_temp}: Temperature of the brake discs for each wheel.
    \item \textbf{current\_time\_str}: Current lap time as a string (e.g., "00:01:23.456").
    \item \textbf{normalized\_car\_position}: Normalized position on the track (0.0 = start, 1.0 = end).
    \item \textbf{wind\_speed}: Wind velocity in meters per second.
    \item \textbf{wind\_direction}: Wind direction in degrees (from north).
\end{itemize}
